# 멋쟁이사자처럼 PBL - 기획 트랙

## 자기소개
- 이름: 이태호
- 파트: 기획 (Product Manager)
- 관심 분야: 학습, 경험

## PM 강의를 듣고 배운 점
- PM은 프로젝트가 잘 흘러가도록 방향을 잡고, 팀원들의 역할과 일정을 조율하는 중요한 역할이라고 느꼈다.
단순히 관리만 하는 사람이 아니라, 문제를 빠르게 파악하고 팀이 같은 목표를 향해 나아가도록 돕는 사람이라고 생각했다.
이번 과제를 통해 PM은 소통과 책임감이 특히 중요한 역할이라는 점을 알게 되었다.

## 이번 학기 목표
- 이번 학기 목표는 PM으로서 팀원들과 원활하게 소통하며 프로젝트가 계획한 방향대로 잘 진행되도록 이끄는 것이다. 또한 일정 관리와 역할 분담을 체계적으로 하여 팀 전체가 효율적으로 협업할 수 있도록 돕고 싶다. 프로젝트를 통해 책임감과 문제 해결 능력도 함께 키우는 것이 목표이다.

## 협업에서 중요한 점
- 소통과 책임감이 중요하다고 생각한다.

## 이번 활동을 통해 배우고 싶은 점
- 일정 관리와 역할 조율 능력을 키우고 싶다.
